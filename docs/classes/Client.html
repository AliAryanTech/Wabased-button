command

  • command(opts, code?): undefined | Collection<number | CommandOptions, any>

  • Create a new command.

    Parameters

    • opts: string | CommandOptions

      Command options object or command name string.

    • Optional code: ((ctx) => Promise<any>)

      If the first parameter is a command name as a string, then you should create a callback function in second parameter.

        • (ctx): Promise<any>

        • Parameters

          • ctx: Ctx

          Returns Promise<any>

    Returns undefined | Collection<number | CommandOptions, any>

    Example

    bot.command('ping', async(ctx) => ctx.reply({ text: 'Pong!' }));

    // same as

    bot.command({
        name: 'ping',
        code: async(ctx) => {
            ctx.reply('Pong!');
        }
    });
